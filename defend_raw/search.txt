Що таке квадратурна формула і для чого вона використовується?
Квадратурна формула використовується для чисельного інтегрування і має вигляд ∫p(x)f(x)dx ≈ ∑A_k f(x_k), де A_k - квадратурні коефіцієнти, x_k - квадратурні вузли.


Яка основна ідея методу Данилевського для знаходження власних значень матриці?
Метод Данилевського зводить вихідну матрицю до нормальної форми Фробеніуса за допомогою подібних перетворень, що спрощує обчислення характеристичного полінома.


Як формується система лінійних рівнянь при інтерполяції многочленом Лагранжа?
При інтерполяції многочленом Лагранжа формується система рівнянь вигляду ∑c_k Φ_k(x_i) = y_i, де c_k - шукані коефіцієнти, Φ_k(x) - базисні функції, (x_i, y_i) - вузли інтерполяції.


Що таке метод прогонки і для яких систем рівнянь він застосовується?
Метод прогонки застосовується для розв'язання систем лінійних рівнянь з трідіагональною матрицею. Він складається з прямого ходу (обчислення прогоночних коефіцієнтів) та зворотного ходу (обчислення розв'язку).


Яка умова збіжності ітераційного процесу в методі простої ітерації для СЛАР?
Умова збіжності методу простої ітерації: норма матриці ітерацій повинна бути менша за 1, тобто ||C|| < 1, де x = Cx + d - ітераційна форма СЛАР.


Що таке явна і неявна схеми в чисельних методах розв'язання диференціальних рівнянь?
Явна схема обчислює нове значення функції безпосередньо з попередніх значень. Неявна схема вимагає розв'язання рівняння на кожному кроці для знаходження нового значення.


Як формулюється теорема Гюа про необхідну умову дійсності всіх коренів алгебраїчного рівняння?
Теорема Гюа стверджує, що якщо всі корені алгебраїчного рівняння дійсні, то квадрат кожного некрайнього коефіцієнта більший за добуток двох сусідніх коефіцієнтів.


Що таке метод Ньютона (дотичних) для розв'язання нелінійних рівнянь і як він реалізується?
Метод Ньютона для розв'язання рівняння f(x)=0 використовує ітераційну формулу x_(n+1) = x_n - f(x_n)/f'(x_n), де f'(x) - похідна функції. Метод вимагає початкового наближення і обчислення похідної на кожній ітерації.


Яка основна ідея методу найменших квадратів при апроксимації функцій?
Метод найменших квадратів мінімізує суму квадратів відхилень між експериментальними даними та значеннями апроксимуючої функції, що дозволяє знайти оптимальні параметри апроксимації.


Що таке сплайн-інтерполяція і в чому її переваги перед поліноміальною інтерполяцією?
Сплайн-інтерполяція використовує кусково-поліноміальні функції для інтерполяції даних. Її перевага - менша схильність до осциляцій при інтерполяції складних функцій, особливо на великих наборах даних.


Як формулюється задача Коші для звичайного диференціального рівняння?
Задача Коші для ЗДР формулюється як знаходження розв'язку y(x) рівняння y' = f(x,y) з початковою умовою y(x0) = y0, де x0 - початкова точка, y0 - початкове значення функції.


Що таке метод Гауса з вибором головного елемента і для чого він використовується?
Метод Гауса з вибором головного елемента - це модифікація методу Гауса для розв'язання СЛАР, де на кожному кроці вибирається максимальний за модулем елемент у стовпці або рядку для покращення числової стійкості алгоритму.


Яка основна ідея методу скінченних різниць для розв'язання крайових задач?
Метод скінченних різниць замінює похідні в диференціальному рівнянні їх різницевими аналогами, перетворюючи задачу на систему лінійних алгебраїчних рівнянь, яку можна розв'язати чисельно.


Що таке метод Ейлера для розв'язання звичайних диференціальних рівнянь і який його порядок точності?
Метод Ейлера - це найпростіший чисельний метод розв'язання ЗДР, який використовує формулу y_(n+1) = y_n + h*f(x_n, y_n), де h - крок інтегрування. Порядок точності методу Ейлера - перший.


Як формулюється теорема про існування та єдиність розв'язку задачі Коші для звичайного диференціального рівняння?
Теорема стверджує, що якщо функція f(x,y) неперервна і задовольняє умову Ліпшиця по y в деякій області, то задача Коші y' = f(x,y), y(x0) = y0 має єдиний розв'язок в околі точки x0.


Що таке метод Рунге-Кутта і як він покращує точність порівняно з методом Ейлера?
Метод Рунге-Кутта - це сімейство ітеративних методів для наближеного розв'язання ЗДР. Він покращує точність, обчислюючи проміжні значення функції на кроці інтегрування, що дозволяє досягти вищого порядку точності (наприклад, четвертого для класичного методу RK4).


Як формулюється задача на власні значення для матриці?
Задача на власні значення для матриці A формулюється як пошук таких чисел λ (власних значень) і ненульових векторів x (власних векторів), що задовольняють рівняння Ax = λx.


Що таке метод степеневих ітерацій і для чого він використовується?
Метод степеневих ітерацій використовується для знаходження найбільшого за модулем власного значення матриці та відповідного власного вектора. Він базується на ітеративному застосуванні матриці до початкового вектора.


Як формулюється теорема про відокремлення коренів алгебраїчного рівняння?
Теорема про відокремлення коренів стверджує, що якщо на кінцях відрізка [a,b] значення многочлена мають різні знаки, то на цьому відрізку існує принаймні один корінь рівняння.


Що таке метод Зейделя для розв'язання систем лінійних алгебраїчних рівнянь?
Метод Зейделя - це ітераційний метод розв'язання СЛАР, який відрізняється від методу простої ітерації тим, що при обчисленні нового наближення i-ї змінної використовуються вже обчислені на поточній ітерації значення попередніх змінних.


Яка основна ідея методу скінченних елементів?
Метод скінченних елементів розбиває область розв'язку на малі підобласті (елементи), апроксимує розв'язок на кожному елементі простими функціями, і зводить задачу до розв'язання системи рівнянь для значень функції у вузлах елементів.


Що таке умовне число матриці і як воно впливає на стійкість розв'язку СЛАР?
Умовне число матриці - це міра її обумовленості, яка показує, наскільки сильно можуть змінитися розв'язки системи при малих змінах вхідних даних. Велике умовне число вказує на погану обумовленість і можливу нестійкість розв'язку.


Як формулюється метод Ньютона для систем нелінійних рівнянь?
Метод Ньютона для систем нелінійних рівнянь F(x)=0 використовує ітераційну формулу x_(k+1) = x_k - J^(-1)(x_k)F(x_k), де J - матриця Якобі системи. Метод вимагає обчислення оберненої матриці Якобі на кожній ітерації.


Що таке метод найшвидшого спуску і для яких задач він застосовується?
Метод найшвидшого спуску - це ітераційний метод оптимізації, який шукає мінімум функції, рухаючись у напрямку, протилежному до градієнта. Застосовується для мінімізації функцій багатьох змінних.


Як формулюється теорема Штурма про чередування коренів?
Теорема Штурма дозволяє визначити кількість дійсних коренів многочлена на заданому інтервалі, використовуючи послідовність многочленів Штурма і підраховуючи зміни знаків у цій послідовності на кінцях інтервалу.


Що таке метод Холецького для розв'язання СЛАР і коли він застосовується?
Метод Холецького - це метод розв'язання СЛАР для симетричних додатно визначених матриць, який розкладає матрицю A на добуток LLᵀ, де L - нижня трикутна матриця. Це дозволяє ефективно розв'язати систему двома прогонами.


Як формулюється задача лінійного програмування?
Задача лінійного програмування полягає у знаходженні максимуму або мінімуму лінійної цільової функції при лінійних обмеженнях на змінні. Математично це записується як max/min c'x при умовах Ax ≤ b, x ≥ 0.


Що таке симплекс-метод і для чого він використовується?
Симплекс-метод - це алгоритм розв'язання задач лінійного програмування, який ітеративно покращує розв'язок, рухаючись від одного допустимого базисного розв'язку до іншого, поки не буде досягнуто оптимального розв'язку або встановлено, що задача не має розв'язку.


Як формулюється метод Ньютона-Рафсона для розв'язання нелінійних рівнянь?
Метод Ньютона-Рафсона для розв'язання рівняння f(x)=0 використовує ітераційну формулу x_(n+1) = x_n - f(x_n)/f'(x_n), де f'(x) - похідна функції. Метод збігається квадратично при хорошому початковому наближенні.


Що таке метод Гауса-Зейделя і чим він відрізняється від методу Якобі?
Метод Гауса-Зейделя - це ітераційний метод розв'язання СЛАР, який, на відміну від методу Якобі, використовує вже обчислені на поточній ітерації значення змінних, що часто призводить до швидшої збіжності.


Як формулюється теорема Банаха про нерухому точку?
Теорема Банаха стверджує, що якщо оператор T є стискаючим на повному метричному просторі, то він має єдину нерухому точку, яку можна знайти методом послідовних наближень x_(n+1) = T(x_n).


Що таке метод градієнтного спуску і для яких задач він застосовується?
Метод градієнтного спуску - це ітераційний алгоритм оптимізації першого порядку для знаходження локального мінімуму функції. Він рухається у напрямку, протилежному до градієнта функції, і застосовується для мінімізації диференційовних функцій багатьох змінних.


Як формулюється задача інтерполяції функції?
Задача інтерполяції полягає у побудові функції, яка проходить через задані точки (вузли інтерполяції) і може бути використана для наближення значень вихідної функції між цими точками.


Що таке сплайн і які види сплайнів ви знаєте?
Сплайн - це кусково-поліноміальна функція, яка має певний порядок гладкості в точках з'єднання сегментів. Основні види: лінійні, квадратичні, кубічні сплайни. Кубічні сплайни часто використовуються через їх оптимальні властивості гладкості та мінімізації кривизни.


Як формулюється метод скінченних об'ємів?
Метод скінченних об'ємів - це чисельний метод розв'язання диференціальних рівнянь, який ділить область розв'язку на малі контрольні об'єми, інтегрує рівняння по кожному об'єму і апроксимує потоки через границі об'ємів.


Що таке метод Монте-Карло і в яких задачах він застосовується?
Метод Монте-Карло - це статистичний метод, що використовує випадкові числа для розв'язання складних математичних задач. Застосовується для обчислення інтегралів, оптимізації, моделювання фізичних процесів та в теорії ймовірностей.


Як формулюється теорема Коші-Ліпшиця для звичайних диференціальних рівнянь?
Теорема Коші-Ліпшиця стверджує, що якщо функція f(x,y) неперервна і задовольняє умову Ліпшиця по y в деякій області, то задача Коші y'=f(x,y), y(x0)=y0 має єдиний розв'язок в околі точки x0.


Що таке метод Рітца і для яких задач він застосовується?
Метод Рітца - це варіаційний метод наближеного розв'язання крайових задач для диференціальних рівнянь. Він зводить задачу до мінімізації функціоналу, апроксимуючи розв'язок лінійною комбінацією базисних функцій.


Як формулюється задача Діріхле для рівняння Пуассона?
Задача Діріхле для рівняння Пуассона полягає у знаходженні функції u(x,y), що задовольняє рівняння Δu = f(x,y) всередині області Ω і приймає задані значення на границі області: u|∂Ω = g(x,y).


Що таке метод релаксації і як він застосовується для розв'язання СЛАР?
Метод релаксації - це ітераційний метод розв'язання СЛАР, який модифікує метод Гауса-Зейделя введенням параметра релаксації ω. Це дозволяє прискорити збіжність методу при правильному виборі ω.


Як формулюється теорема про середнє значення для визначених інтегралів?
Теорема про середнє значення стверджує, що для неперервної на [a,b] функції f(x) існує точка c∈[a,b] така, що ∫[a,b]f(x)dx = f(c)(b-a), де f(c) - середнє значення функції на відрізку.


Що таке метод Ньютона-Канторовича і для яких задач він застосовується?
Метод Ньютона-Канторовича - це узагальнення методу Ньютона для нелінійних операторних рівнянь у функціональних просторах. Застосовується для розв'язання нелінійних інтегральних рівнянь та крайових задач для нелінійних диференціальних рівнянь.


Як формулюється принцип максимуму для гармонічних функцій?
Принцип максимуму стверджує, що гармонічна функція в обмеженій області досягає свого максимального і мінімального значення лише на границі області, якщо вона не є константою всюди в області.


Що таке метод скінченних елементів з розривними базисними функціями?
Це варіант методу скінченних елементів, де базисні функції можуть мати розриви на границях елементів. Це дозволяє краще апроксимувати розв'язки з розривними похідними та підвищує точність методу для певних класів задач.


Як формулюється теорема Фредгольма для інтегральних рівнянь другого роду?
Теорема Фредгольма стверджує, що для інтегрального рівняння другого роду з неперервним ядром існує єдиний розв'язок для всіх значень параметра, крім скінченного числа особливих значень, які є власними числами відповідного інтегрального оператора.


Що таке метод продовження за параметром і для яких задач він застосовується?
Метод продовження за параметром - це техніка розв'язання нелінійних рівнянь або систем рівнянь шляхом введення додаткового параметра і поступової зміни цього параметра від простого випадку до бажаного. Застосовується для розв'язання складних нелінійних задач.


Як формулюється теорема Ляпунова про стійкість?
Теорема Ляпунова стверджує, що якщо для системи диференціальних рівнянь існує додатно визначена функція Ляпунова, похідна якої вздовж траєкторій системи є від'ємно визначеною, то положення рівноваги системи є асимптотично стійким.


Що таке метод Галеркіна і як він відрізняється від методу Рітца?
Метод Галеркіна - це проекційний метод розв'язання крайових задач, який вимагає, щоб нев'язка була ортогональна до базисних функцій. На відміну від методу Рітца, він не вимагає існування відповідного функціоналу і може застосовуватися до ширшого класу задач.


Як формулюється теорема Ньютона-Лейбніца?
Теорема Ньютона-Лейбніца стверджує, що якщо f(x) - неперервна функція на [a,b] і F(x) - її первісна, то ∫[a,b]f(x)dx = F(b) - F(a). Ця теорема встановлює зв'язок між визначеним інтегралом і первісною.


Що таке метод стрільби і для яких задач він застосовується?
Метод стрільби - це чисельний метод розв'язання крайових задач для звичайних диференціальних рівнянь, який зводить крайову задачу до послідовності задач Коші. Застосовується, коли крайові умови задані на різних кінцях інтервалу.


Як формулюється теорема Гріна і яке її значення в математичній фізиці?
Теорема Гріна пов'язує подвійний інтеграл по області з криволінійним інтегралом по її границі. У математичній фізиці вона використовується для переформулювання диференціальних рівнянь у частинних похідних в інтегральну форму.


Що таке метод скінченних різниць з підвищеним порядком точності?
Це модифікація стандартного методу скінченних різниць, яка використовує більше точок сітки для апроксимації похідних, що дозволяє досягти вищого порядку точності. Застосовується для підвищення точності чисельних розв'язків диференціальних рівнянь.


Як формулюється принцип Дірак для варіаційних задач?
Принцип Дірак стверджує, що серед усіх можливих шляхів системи реалізується той, для якого дія (інтеграл від лагранжіана) є стаціонарною. Це узагальнення принципу найменшої дії в класичній механіці.


Що таке метод Кеплера і для яких задач він застосовується?
Метод Кеплера - це аналітичний метод розв'язання рівнянь руху в задачі двох тіл. Він використовується в небесній механіці для обчислення орбіт планет і супутників, зводячи нелінійну задачу до розв'язання трансцендентного рівняння Кеплера.


Як формулюється теорема Ліувілля в теорії диференціальних рівнянь?
Теорема Ліувілля стверджує, що для системи диференціальних рівнянь з неперервними частинними похідними фазовий об'єм зберігається вздовж траєкторій системи. Це має важливе значення в статистичній механіці та теорії динамічних систем.


Що таке метод граничних елементів і в чому його переваги перед методом скінченних елементів?
Метод граничних елементів - це чисельний метод розв'язання крайових задач, який зводить задачу до інтегрального рівняння на границі області. Його перевага - зменшення розмірності задачі, що особливо ефективно для задач з нескінченними областями.


Як формулюється теорема Коші про вичети?
Теорема Коші про вичети стверджує, що інтеграл аналітичної функції по замкненому контуру дорівнює 2πi, помноженому на суму вичетів функції всередині контуру. Це потужний інструмент для обчислення комплексних інтегралів.


Що таке метод Гальоркіна з розривними елементами?
Це варіант методу Гальоркіна, де допускаються розриви базисних функцій на границях елементів. Це дозволяє краще апроксимувати розв'язки з сильними градієнтами або розривами і застосовується в задачах механіки суцільних середовищ та гідродинаміки.


Як формулюється теорема Ляпунова про першу нестійкість?
Теорема стверджує, що якщо всі власні значення лінеаризованої системи мають додатні дійсні частини, то положення рівноваги нелінійної системи є нестійким. Це важливий результат для аналізу стійкості нелінійних систем.


Що таке метод Гіра для жорстких систем диференціальних рівнянь?
Метод Гіра - це сімейство неявних багатокрокових методів для чисельного розв'язання жорстких систем звичайних диференціальних рівнянь. Він забезпечує високу стабільність і ефективність для задач з широким діапазоном часових масштабів.


Як формулюється теорема Каратеодорі про існування розв'язку задачі Коші?
Теорема Каратеодорі надає менш обмежувальні умови для існування розв'язку задачі Коші, ніж теорема Пікара. Вона стверджує, що розв'язок існує, якщо права частина рівняння є неперервною за x і задовольняє умову Каратеодорі за y.


Що таке метод Ньютона-Крилова і для яких задач він застосовується?
Метод Ньютона-Крилова - це варіант методу Ньютона для розв'язання нелінійних систем рівнянь великої розмірності. Він комбінує ідеї методу Ньютона з ітераційними методами підпростору Крилова для ефективного розв'язання лінеаризованих систем.


Як формулюється теорема Фробеніуса-Перрона для невід'ємних матриць?
Теорема стверджує, що для невід'ємної нерозкладної матриці існує додатне власне значення, яке є простим і не менше за модулем від усіх інших власних значень. Відповідний власний вектор має строго додатні компоненти.


Що таке метод колокації і як він відрізняється від методу Галеркіна?
Метод колокації - це чисельний метод розв'язання диференціальних та інтегральних рівнянь, де вимагається точне виконання рівняння в заданих точках (точках колокації). На відміну від методу Галеркіна, він не використовує інтегральні співвідношення.


Як формулюється теорема Ліувілля-Остроградського?
Теорема Ліувілля-Остроградського (теорема про дивергенцію) пов'язує потрійний інтеграл від дивергенції векторного поля по об'єму з потоком цього поля через замкнену поверхню, що обмежує об'єм. Це узагальнення теореми Гріна на тривимірний випадок.


Що таке метод Ньютона-Канторовича для операторних рівнянь?
Метод Ньютона-Канторовича - це узагальнення методу Ньютона на випадок нелінійних операторних рівнянь у функціональних просторах. Він застосовується для розв'язання нелінійних інтегральних рівнянь та нелінійних крайових задач.


Як формулюється теорема Пуанкаре-Бендіксона?
Теорема Пуанкаре-Бендіксона описує поведінку траєкторій автономної системи диференціальних рівнянь на площині. Вона стверджує, що обмежена траєкторія, яка не прямує до точки рівноваги, має наближатися до замкненої орбіти.


Що таке метод декомпозиції області і для яких задач він застосовується?
Метод декомпозиції області - це техніка розв'язання крайових задач, яка розбиває область на підобласті, розв'язує задачі на підобластях і узгоджує розв'язки на границях. Застосовується для паралельних обчислень та розв'язання задач у складних геометріях.


Як формулюється теорема Гурса про аналітичне продовження?
Теорема Гурса стверджує, що якщо дві аналітичні функції збігаються на множині, що має граничну точку в області їх визначення, то вони збігаються у всій цій області. Це важливий результат для теорії аналітичних функцій.
