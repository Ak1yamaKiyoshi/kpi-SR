
1. Яка основна відмінність між методом Сімпсона та квадратурною формулою Гауса?
Метод Сімпсона базується на інтерполяції функції квадратичним поліномом з рівновіддаленими вузлами. Квадратурна формула Гауса використовує спеціально підібрані вузли (корені поліномів Лежандра), що дозволяє досягти вищої точності при тій самій кількості обчислень функції. Формула Гауса точна для поліномів вищого степеня, ніж формула Сімпсона при однаковій кількості вузлів.

2. Як оцінюється похибка в методі Сімпсона та в квадратурній формулі Гауса? Чому ці оцінки відрізняються?
Для методу Сімпсона похибка оцінюється як:
|R_4(f)| ≤ ((b-a)^5 / (180n^4)) * max|f^(4)(x)|
Для квадратурної формули Гауса:
|R_m(f)| ≤ ((m!)^4 * (b-a)^(2m+1)) / ((2m+1) * [(2m)!]^3) * max|f^(2m)(ξ)|
Оцінки відрізняються, бо формула Гауса має вищий порядок точності. Вона залежить від 2m-ої похідної, тоді як метод Сімпсона - від 4-ої.

3. Як вибір вузлів інтегрування впливає на точність квадратурних формул?
Вибір вузлів суттєво впливає на точність. Рівновіддалені вузли (як у методі Сімпсона) прості у використанні, але менш ефективні. Вузли Гауса (корені поліномів Лежандра) оптимально розташовані для максимізації точності інтегрування. Це дозволяє формулі Гауса бути точною для поліномів вищого степеня при тій самій кількості вузлів.

4. Чому в методі Гауса використовуються поліноми Лежандра? Яке їх значення для побудови квадратурної формули?
Поліноми Лежандра використовуються через їх ортогональність на відрізку [-1, 1]. Це дозволяє:
1) Спростити систему рівнянь для знаходження коефіцієнтів та вузлів формули.
2) Забезпечити максимальну точність формули для поліномів заданого степеня.
3) Отримати оптимальне розташування вузлів інтегрування.

5. Як реалізується адаптивний вибір кількості вузлів у методі Гауса в наданому коді?
У коді реалізовано адаптивний вибір кількості вузлів m для методу Гауса:
1) Починають з m = 2.
2) Оцінюють похибку за формулою (1.14).
3) Якщо похибка перевищує задану точність, m збільшують на 1.
4) Процес повторюється, доки не буде досягнута потрібна точність.
Це дозволяє автоматично підібрати оптимальну кількість вузлів для заданої точності.
