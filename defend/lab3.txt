Згідно з наданими теоретичними матеріалами та кодом, ось список складних запитань з детальними відповідями:

1. Яка основна відмінність між методом простої ітерації (Якобі) та методом Зейделя?

Основна відмінність полягає у способі обчислення нових наближень. У методі Якобі всі компоненти нового наближення обчислюються на основі попереднього, тоді як у методі Зейделя вже обчислені компоненти нового наближення одразу використовуються для розрахунку наступних компонент.

2. Яка умова має виконуватися для матриці C у методі простої ітерації?

Для матриці C має виконуватися умова: 
α = max(Σ|cij|) < 1, де сума береться або по рядках, або по стовпцях, або за всіма елементами матриці. Ця умова забезпечує збіжність ітераційного процесу.

3. Як привести систему до вигляду з діагональною перевагою, якщо вхідна матриця не має такої властивості?

Для приведення системи до вигляду з діагональною перевагою можна використати еквівалентні перетворення. Наприклад, можна виконати один крок методу Гауса, комбінуючи рядки для отримання нульового недіагонального елемента у вибраному стовпці.

4. Як формується матриця C та вектор d у методі простої ітерації?

Матриця C формується наступним чином:
cij = -aij/aii, якщо i ≠ j
cij = 0, якщо i = j
Вектор d формується як:
di = bi/aii

5. Який критерій використовується для завершення ітераційного процесу в обох методах?

Критерій завершення ітераційного процесу:
max|xj(k+1) - xj(k)| < ε, 
де ε - задана точність, k - номер ітерації.

6. Як обчислюється вектор нев'язки і яке його значення для оцінки точності розв'язку?

Вектор нев'язки обчислюється як r = b - Ax, де x - отриманий розв'язок. Він показує, наскільки отриманий розв'язок задовольняє вихідну систему рівнянь. Чим ближче вектор нев'язки до нульового вектора, тим точніший розв'язок.

7. Як реалізується один крок методу Гауса для приведення матриці до діагональної переваги у наданому коді?

У функції gaussian_elimination_one_step спочатку знаходиться рядок з максимальним за модулем елементом у вибраному стовпці. Потім цей рядок міняється місцями з поточним, щоб максимальний елемент опинився на діагоналі. Далі виконується віднімання поточного рядка від усіх нижчих рядків з відповідними коефіцієнтами для обнуління елементів під діагоналлю у вибраному стовпці.
